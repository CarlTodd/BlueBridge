# P6628 [省选联考 2020 B 卷] 丁香之路

## 题目描述

春暖花开，万物复苏，随着疫情的逐渐过去，Yazid 带着他的 $n$ 个好朋友来到 T 大校园参观游览。方便起见，我们将他们从 $1$ 至 $n$ 编号。

T 大校园的版图可以抽象成一张 $n$ 个顶点的无向图（顶点编号从 $1$ 至 $n$）。且对于任意两个不同顶点，设它们的编号分别为 $i, j(i\neq j)$，则它们之间有一条需要花费 $|i - j|$ 单位时间通过的无向边。

丁香花是 T 大的校花之一。时下正值丁香花盛开之际，校园内的 $m$ 条道路上都开有丁香花。Yazid 的朋友们对丁香花十分感兴趣，因此他们都希望遍历**所有**开有丁香花的 $m$ 条道路。

Yazid 的朋友们从顶点 $s$ 出发。其中，第 $i$ 个朋友希望以顶点 $i$ 为终点终止他的参观。与此同时，如上面所述，每个朋友都必须经过开着丁香花的 $m$ 条道路各至少一次。

Yazid 的朋友不想太过疲累，因此他们希望花尽可能少的时间来完成他们的目标。

请你计算 Yazid 的朋友们分别需要花费多少单位时间完成他们的目标。

你现在的思路其实非常接近“正解”，但你对这道题的本质理解有一点点**偏差**。

这道题其实不是一个纯粹的“欧拉路径”存在性判断题，而是一个经典的**中国邮递员问题（Chinese Postman Problem）**的变种。

### 为什么不能直接套用 Hierholzer？

1.  **“至少经过一次”**：题目要求所有 $m$ 条边“至少”经过一次。这意味着你可以**重复走某些边**。
2.  **补全图的连通性/欧拉性**：为了让图满足欧拉路径的要求，你需要通过重复走边，把图中所有点的度数调整为满足欧拉路径的条件（即：除了起点 $s$ 和终点 $t$ 以外，所有点度数为偶数）。
3.  **计算的是“时间（费用）”**：题目问的是最少花费的时间，而不是让你把路径打出来。如果你跑 Hierholzer，你需要先通过重复走边构造出一张“欧拉图”，然后再跑。但实际上，我们可以通过数学方法直接计算出最少需要“补充”多少费用。

### 正确的解题思路：贪心 + 奇偶度数配对

这道题利用了题目给出的图的一个特殊性质：**任意两个点 $i, j$ 之间都有边，且边权为 $|i - j|$**。这是一个非常有用的线性结构。

#### 核心步骤：

1.  **统计原始度数**：
    先统计这 $m$ 条“有丁香花”的边在每个点上的度数。记为 `deg[i]`。

2.  **确定“奇度数”集合**：
    找出所有度数为奇数的点，放入一个集合 $O$ 中。
    *   在标准的欧拉回路/路径问题中，如果一个图有奇度数点，它是无法直接走完的。
    *   为了走完，我们需要给某些点之间“增加连线”（即重复走边），从而把它们的度数变偶。

3.  **处理起点 $s$ 和终点 $t$**：
    *   假设你当前要去的终点是 $t$。
    *   要形成一条从 $s$ 到 $t$ 的欧拉路径，满足条件的充要条件是：
        *   $s$ 和 $t$ 的度数是奇数，其他所有点的度数都必须变成偶数。
   

4.  **贪心配对（关键！）**：
    *   现在集合 $O'$ 里全是需要变成偶数的点。我们需要两两配对，把它们连起来。
    *   由于图是线性的（边权为 $|i - j|$），对于集合 $O'$ 中的点，**最优的配对策略是将排好序的点相邻配对**。
    *   例如：$O' = \{p_1, p_2, p_3, p_4\}$，且 $p_1 < p_2 < p_3 < p_4$。
    *   最小花费 = $(p_2 - p_1) + (p_4 - p_3)$。
    *   这是一个经典的结论：在数轴上，两两配对使得距离之和最小，就是排序后相邻配对。

5.  **总费用公式**：
    **总时间 = (所有 $m$ 条边的原始边权和) + (配对产生的额外时间)**

### 总结你的算法流程

不需要跑 DFS，不需要跑 Hierholzer，只需要：

1.  **预处理**：计算 $m$ 条边的权值和 `sum_weights`，统计每个节点的度数 `deg[]`。
2.  **对于每个 $i$ (即终点 $t$)**：
    *   拷贝一份原始的奇度数点集合。
    *   将 $s$ 和 $t$ 的状态在该集合中翻转（在集合里的移出，不在的加入）。
    *   对这个集合排序。
    *   两两相邻计算差值求和。
    *   最终答案 = `sum_weights` + `上述差值和`。

### 为什么不用跑 Hierholzer？
因为题目只要求**计算花费**，不要求输出具体的路径。如果题目要求输出路径，你才需要跑 Hierholzer 算法。只要你把路径上所有“重复走”的边都补齐了，原图就变成了一张完美的欧拉图，此时 Hierholzer 必定能跑通。

这个做法的时间复杂度非常低，只需 $O(N \log N)$ 甚至更优，完美解决这道题。这就是“中国邮递员问题”在特殊图结构下的解法。
